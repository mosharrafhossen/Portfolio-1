আমার সম্পর্কে

আমি মোশাররফ হোসেন, একজন উত্সাহী ওয়েব ডেভেলপার ও ডিজাইনার। আমার লক্ষ্য হলো ব্যবহারকারীদের জন্য সুন্দর, দ্রুত এবং রেসপন্সিভ ওয়েবসাইট তৈরি করা। আমি সবসময় নতুন প্রযুক্তি শিখতে আগ্রহী এবং ক্রিয়েটিভ প্রজেক্টে কাজ করতে পছন্দ করি।

দক্ষতা

  • HTML5 & CSS3: semantic এবং clean কোডিং
  • Responsive Design: মোবাইল, ট্যাবলেট ও ডেস্কটপে সুন্দর লেআউট
  • JavaScript (vanilla): ইন্টারঅ্যাকটিভ ওয়েব ফিচার তৈরি
  • UI/UX Design: ব্যবহারকারী-বান্ধব ইন্টারফেস ডিজাইন
  • Version Control: Git & GitHub দিয়ে কোড ম্যানেজমেন্ট

অভিজ্ঞতা

আমি বিভিন্ন প্রজেক্টে কাজ করেছি, যার মধ্যে রয়েছে ব্যক্তিগত ওয়েবসাইট, ক্লায়েন্ট ওয়েবসাইট, এবং ছোট অনলাইন অ্যাপ্লিকেশন। ক্লায়েন্টদের চাহিদা অনুযায়ী কাস্টম ওয়েব সলিউশন ডেভেলপ করেছি।

  • Freelance Projects: ৫+ ওয়েবসাইট ক্লায়েন্টদের জন্য তৈরি
  • Team Collaboration: টিমের সাথে মিলিতভাবে প্রজেক্ট সম্পন্ন
  • Problem Solving: কোড এবং ডিজাইন সংক্রান্ত সমস্যার দ্রুত সমাধান

সেবা

আমি বিভিন্ন ওয়েব ডেভেলপমেন্ট ও ডিজাইন সেবা প্রদান করি যা ক্লায়েন্টদের প্রয়োজন অনুযায়ী তৈরি। আমার মূল লক্ষ্য হলো ব্যবহারকারী-বান্ধব, রেসপন্সিভ এবং আধুনিক ওয়েবসাইট ডেভেলপ করা।

ওয়েব ডিজাইন

UI/UX ডিজাইন, প্রোটোটাইপিং এবং ভিজ্যুয়াল কনসেপ্ট তৈরি। আধুনিক ট্রেন্ড অনুসারে আকর্ষণীয় এবং ব্যবহারকারী-বান্ধব ডিজাইন নিশ্চিত করি।

  • Figma/Adobe XD ডিজাইন তৈরি
  • ইউজার ফ্রেন্ডলি ইন্টারফেস
  • Creative Layout & Typography

ফ্রন্টএন্ড ডেভেলপমেন্ট

HTML, CSS এবং JavaScript ব্যবহার করে দ্রুত এবং কার্যকর ওয়েব পেজ তৈরি। প্রয়োজন অনুযায়ী ইন্টারঅ্যাকটিভ ফিচার ও এনিমেশন যোগ করা।

  • Responsive Layout & Cross-browser Compatibility
  • Vanilla JavaScript & DOM Manipulation
  • Performance Optimized Code

  • Flexible Grid & Layout
  • Touch-friendly Elements
